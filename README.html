Jingooth — Asset Pack 12 เลเวล (512×512)

แพ็กนี้เป็น ชุดภาพ + โครง สำหรับใช้กับเกมสไตล์ Suika โดยไม่ต้องแก้โค้ด — เปลี่ยนไฟล์ภาพแล้ว Deploy ได้เลย

ไฟล์ในแพ็ก

  • /images/face1.png ... face12.png — ตัวอย่างภาพวงกลมสี (โปร่งใส 512×512)
  • config.json — ระบุลำดับ 12 เลเวล
  • index.html — หน้า Preview/สตาร์ท

วิธีใช้งานแบบ “ไม่ต้องโค้ด”

  1. เตรียมรูปหน้าคน 12 ไฟล์ PNG โปร่งใส 512×512 ตั้งชื่อ face1.png ถึง face12.png
  2. ทางเลือก A (รวดเร็ว): ใช้โปรเจกต์แม่แบบ (เช่นคลัง Suika บน GitHub)
    1. Fork โปรเจกต์แม่แบบ (เช่น daxigua หรือคลัง Suika ใดๆ)
    2. แทนที่โฟลเดอร์ภาพของโปรเจกต์นั้นด้วย /images จากแพ็กนี้
    3. ถ้ามีไฟล์คอนฟิกของโปรเจกต์ ให้ตรวจว่าลำดับรูป 1→12 ตรงชื่อไฟล์
    4. Push ขึ้น GitHub
    5. ไปที่ Vercel → New Project → Import จาก GitHub → Deploy
  3. ทางเลือก B (ใช้แพ็กนี้เป็น Static Site แสดงตัวอย่าง):
    1. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub เป็น repo ชื่อ jingooth
    2. บน Vercel → Import โปรเจกต์ jingooth → Deploy
    3. หน้านี้จะเป็น Preview ของภาพ 12 เลเวล (ไม่ใช่เอ็นจิ้นเกม) — ใช้เพื่อทดสอบ/รีวิวรูปได้

เคล็ดลับคุณภาพ

  • เว้นขอบรอบรูป 5–8% เพื่อลดการชนขอบ/ทับกัน
  • ให้รูปอยู่ในทรงกลม/โค้งมน เพื่อให้ hitbox ในเกมลื่นขึ้น
  • ใช้โทนสี/สไตล์ใกล้กันทั้ง 12 เลเวล

การเปลี่ยนชื่อแสดงผล

แก้ไขชื่อบนหน้าเว็บใน index.html ได้ตามต้องการ หรือใช้ค่า projectName ใน config.json.
